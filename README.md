## 📱 アプリについて

OKN Strip は、視運動性眼振（Optokinetic Nystagmus）を利用した神経トレーニングアプリです。動くストライプパターンを見つめることで、視覚-運動統合機能を刺激します。

## 🧠 神経科学的背景

視運動性眼振（OKN）トレーニングは以下の神経システムを活性化します：

- **前庭系** - 平衡感覚と空間認識
- **小脳** - 運動協調と眼球運動制御
- **脳幹** - 視覚反射と自律神経機能
- **視覚野** - 動体視力と周辺視野処理

中央の視点を固視しながら、周辺視野で動くストライプを感じることで、これらの神経回路を効果的に刺激できます。

## ✨ 機能

| 機能 | 説明 |
|------|------|
| 🔄 方向選択 | 左右の移動方向を切り替え |
| 🎨 色の変更 | 黒、赤、アイコンパターンから選択 |
| ⚡ 速度調整 | 1〜10段階で速度を調整 |
| 📏 幅調整 | ストライプの幅を30〜120pxで調整 |
| 🎯 視点マーカー | 中央の固視点を表示/非表示 |
| 🖥️ フルスクリーン | UIを非表示にして集中モード |

## 📲 インストール方法

### iOS (Safari)
1. Safari でアプリを開く
2. 共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択

### Android (Chrome)
1. Chrome でアプリを開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択

### PC (Chrome/Edge)
1. ブラウザでアプリを開く
2. アドレスバーのインストールアイコンをクリック
3. 「インストール」を選択

## 🚀 使い方

1. **開始**: ▶ボタンをタップしてストライプの動きを開始
2. **固視**: 中央の緑色の十字マーカーを見つめる
3. **感覚**: 周辺視野でストライプの動きを感じる
4. **調整**: 設定メニューで速度や方向を調整
5. **停止**: ⏸ボタンで一時停止

## ⚠️ 注意事項

- **光過敏症の方は使用を控えてください**
- **てんかんの既往歴がある方は医師に相談してください**
- 気分が悪くなった場合はすぐに使用を中止してください
- 1セッション5〜10分程度を目安にしてください

## 🛠️ 技術仕様

- **PWA対応** - オフラインで動作可能
- **レスポンシブ** - スマートフォン・タブレット・PC対応
- **軽量** - 高速読み込み・低消費電力
